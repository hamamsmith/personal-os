# Project Specific Rules

- **Auto Git Push**: Setiap kali ada perubahan kodingan, script, atau file apapun di project ini, Antigravity WAJIB langsung menjalankan perintah `git add .`, `git commit`, dan `git push` secara otomatis ke repository. Dilarang menunggu aba-aba atau bertanya dulu kepada user.

- **Auto Backup Chat**: Setiap kali selesai berdiskusi atau melakukan task perbaikan, Antigravity WAJIB merangkum inti obrolan dan pencapaiannya lalu menyimpannya ke file `C:\Users\Smith\Documents\Chat_Antigravity\habit_tracker_backup.md` (gunakan PowerShell untuk create/append ke file). Hal ini wajib dilakukan supaya riwayat diskusi tetep aman biarpun sistem ke-reset, dan pastinya 100% aman dari repo Git lu.
